# Furniture Home UI

Minimal premium ecommerce homepage for tables and furniture.

## Included Sections

- Sticky `Header` with search, wishlist, cart, profile, mobile menu, and bottom mobile nav
- `HeroSection` with CTA buttons and gradient overlay
- `CategoryCard` grid for featured categories
- `ProductGrid` + `ProductCard` for featured and best seller products
- `PromoBanner` promotional blocks
- `WhyChooseUsSection` value proposition cards
- `TestimonialSection` customer reviews
- `NewsletterSection` subscription block
- Dark `Footer` with links and payment methods

## Data Source

Edit `src/components/furniture-home/data.ts` to change categories, products, banners, or testimonials.

## Quick Run

```bash
npm run dev
```

## Validation

npm run lint
